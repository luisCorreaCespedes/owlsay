### OWLSAY

> OWLSAY is a talking owl with configurable messages

## Install

To install this package just type and execute:

```sh
npm install -g owlsay
```

or

```sh
npm i owlsay
```

## How to use:

To use this package, write:

```js
const example = require('owlsay')
```
